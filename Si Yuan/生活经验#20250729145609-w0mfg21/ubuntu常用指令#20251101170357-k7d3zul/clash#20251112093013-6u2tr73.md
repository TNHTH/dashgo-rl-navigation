* 执行 `./clash -d .` 即可启动 Clash，同时启动 HTTP 代理和 Socks5 代理。

  如提示权限不足，请执行 `chmod +x clash`

  ![](https://small-files.make-w0rld-static.club/file/ikuuu-static-release/ikuuu-web-front/theme-1.0.9/theme-e5556/malio/img/tutorial/linux-clash-3.jpg)
* 4.访问 [Clash Dashboard ](https://clash.razord.top/) 可以进行切换节点、测延迟等操作。

  Host: `127.0.0.1`，端口: `9090`

  ![](https://small-files.make-w0rld-static.club/file/ikuuu-static-release/ikuuu-web-front/theme-1.0.9/theme-e5556/malio/img/tutorial/linux-clash-4.jpg)
* 5.以 Ubuntu 19.04 为例，打开系统设置，选择网络，点击网络代理右边的 ⚙ 按钮，选择手动，填写 HTTP 和 HTTPS 代理为 `127.0.0.1:7890`，填写 Socks 主机为 `127.0.0.1:7891`，即可启用系统代理。

  ![](https://small-files.make-w0rld-static.club/file/ikuuu-static-release/ikuuu-web-front/theme-1.0.9/theme-e5556/malio/img/tutorial/linux-clash-5.jpg)
