## 🎯 核心结论速查

### MBTI类型
```
INTJ（建筑师型）- 战略家+知识库特化
- Ni主导：战略洞察、风险识别、未来模拟
- Te辅助：目标导向、风险管理、系统创造力不足
- Fi第三：自我觉察、观点表达、过度自我批评
- Si特殊：知识保存、建立知识库（INTJ中少见）
- Se劣势：执行力严重不足，需要紧急发展
```

## 🔧 文件管理原则

### 命名规范
- 格式：`主题_类型_YYYY-MM-DD.md`
- 例如：`MBTI深度分析报告_郭伟浩_2026-01-18.md`
- 日期为创建日期或最后更新日期

### 版本管理
- 当有重大新理解时，创建新文档（保留旧版）
- 小幅更新直接修改原文档
- 更新文档底部的"最后更新"时间

### 归档原则
- 超过6个月的文档移入 `archive/` 文件夹
- 有更新版本的旧文档移入 `archive/`
- `archive/` 按年份组织（archive/2026/、archive/2027/）

### 删除原则
- **不删除**任何分析文档（都是成长记录）
- 重复内容可合并后保留一个
- 临时笔记可删除（但需谨慎）
