或

或

```

**file-organizer skill 会自动**：

3. 检查文件组织
4. 自动清理并整理

### 方式2：作为日常习惯

**养成习惯**：每次对话结束前说一句：

```

```

这样你的文件系统会始终保持整洁。

**每周一次**（建议周末）：

```

file-organizer 会：
- 清理一周积累的临时文件
- 检查是否有新的重复文件
- 验证文件分类是否正确
