## 📋 命名规范

### 文件命名格式
```
[主题]_[说明]_v[版本号]_[YYYY-MM-DD].[扩展名]
```

### 示例
- `个人综合分析_郭伟浩_v1.0_2026-01-18.md`
- `基金组合深度分析_v1.0_2026-01-14.md`
- `心理咨询Agent_改造方案_v2.0_2026-01-20.md`

### 版本号规则
- `v1.0` - 初始版本
- `v2.0` - 重大更新
- `v2.5` - 小版本更新
- `v2.5.1` - 修正版本
